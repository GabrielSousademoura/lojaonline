# 🛒 Marketplace Data Simulator — Power BI Portfolio Project

Banco de dados simulado de um marketplace C2C (estilo OLX / Enjoei) criado para fins analíticos e desenvolvimento de habilidades em Business Intelligence.

##  Sobre o projeto

Este projeto gera e estrutura dados fictícios de uma plataforma de compras e vendas entre pessoas físicas, cobrindo todo o ciclo da transação — do cadastro do usuário ao fechamento do pedido. O objetivo é ter uma base realista o suficiente para construir dashboards ricos e explorar recursos avançados do Power BI.

##  Objetivos

- Praticar modelagem dimensional
- Popular um banco de dados relacional com dados consistentes e volumosos
- Desenvolver relatórios e dashboards analíticos no Power BI
- Aplicar boas práticas de ETL e transformação de dados com Power Query

##  Escopo dos dados simulados

- Usuários (compradores e vendedores)
- Anúncios de produtos com categorias, preços e status
- Transações, pedidos e histórico de pagamentos
- Avaliações e reputação de usuários
- Localização geográfica (estados / cidades)

##  Tecnologias utilizadas

- **Banco de dados:** [PostgreSQL]
- **Geração de dados:** [Python / SQL scripts]
- **BI & Visualização:** Microsoft Power BI

##  Dashboards previstos

- Visão geral de vendas por período e categoria
- Ranking de vendedores e produtos mais anunciados
- Funil de conversão (anúncio → pedido → pagamento)
- Distribuição geográfica de usuários e transações
